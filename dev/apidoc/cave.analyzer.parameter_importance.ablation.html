cave.analyzer.parameter_importance.ablation module¶

class cave.analyzer.parameter_importance.ablation.Ablation(runscontainer, marginal_threshold=0.05)[source]¶

Bases: cave.analyzer.parameter_importance.base_parameter_importance.BaseParameterImportance

Ablation Analysis is a method to determine parameter importance by comparing two parameter configurations, typically the default and the optimized configuration. It uses a greedy forward search to determine the order of flipping the parameter settings from default configuration to incumbent such that in each step the cost is maximally decreased.

Calculate parameter-importance using the PIMP-package.

plot_bokeh()¶

This function should recreate the bokeh-plot from scratch with as little overhead as possible. This is needed to show the bokeh plot in jupyter AND save it to the webpage. The bokeh plot needs to be recreated to be displayed in different outputs for reasons beyond out control. So save all analysis results in the class and simply redo the plotting with this function. This function needs to be called if bokeh-plots are to be displayed in notebook AND saved to html-result.
